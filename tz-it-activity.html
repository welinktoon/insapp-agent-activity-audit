Правки страницы IT-деятельность Insapp

На скриншотах отмечены места, где нужно внести изменения на странице https://insapp.ru/it-deyatelnost/: верхний блок с реквизитами, блок описания услуг и нижний блок стоимости.

1Верх страницы: добавить блок «О сайте»

Над реквизитами добавить отдельный информационный блок про принадлежность сайта.

Сейчас Абзац про официальный сайт находится внутри блока с реквизитами.
Заменить на Добавить отдельный блок «О сайте» с текстом: «Сайт insapp.ru принадлежит ООО “ИНСАП” и используется для размещения информации о программном комплексе “Инсап”, его функциональных возможностях, условиях предоставления и контактных данных правообладателя».

2Верх страницы: переименовать заголовок

В блоке реквизитов поменять только заголовок.

Сейчас Заголовок блока: «Реквизиты».
Заменить на Заголовок блока: «Реквизиты и контакты».

3Верх страницы: убрать абзац из реквизитов

Текст про официальный сайт не должен дублироваться в реквизитах после добавления блока «О сайте».

Сейчас Внутри блока «Реквизиты» есть абзац про официальный сайт insapp.ru.
Заменить на Убрать этот абзац из блока реквизитов. Его смысл остается только в новом блоке «О сайте».

4Верх страницы: добавить email

В реквизиты и контакты добавить электронную почту.

Сейчас В блоке реквизитов нет отдельной строки с электронной почтой.
Заменить на После строки с КПП добавить строку: «Электронная почта: support@insapp.ru».

5Описание услуг: переименовать заголовок

Поменять название блока на более точное для страницы IT-деятельности.

Сейчас Заголовок блока: «Описание услуг».
Заменить на Заголовок блока: «Функциональные возможности ПО».

6Описание услуг: заменить текст блока

Коммерческое описание заменить на описание функций программного комплекса.

Сейчас Текст блока выглядит как описание коммерческих услуг.
Заменить на Текст блока: «Программный комплекс “Инсап” помогает подключать на сайты и в приложения онлайн-расчеты, формы заявок, виджеты, API-интеграции, личный кабинет и статусы обращений. ПО обеспечивает прием, обработку и передачу структурированных данных между пользователем сайта и подключенными организациями».

7Стоимость: добавить блок на странице

Стоимость должна быть указана на странице IT-деятельности.

Сейчас Отдельного блока стоимости на странице нет.
Заменить на Добавить на этой странице отдельный блок «Стоимость» с текстом: «Стоимость лицензии на использование программного комплекса “Инсап” - от 50 000 рублей в месяц».

8Стоимость: убрать ссылку на /partner/

В блоке стоимости не должно быть перехода на партнерскую страницу.

Сейчас Для стоимости используется ссылка на /partner/.
Заменить на Убрать ссылку на /partner/ из этого места. Стоимость показывать прямо на странице /it-deyatelnost/.

9Права на ПО

В блоке про способы предоставления уточнить SaaS бизнес-языком.

Сейчас В блоке про способы предоставления используется короткая формулировка «SaaS».
Заменить на Использовать формулировку: «программное обеспечение как услуга (SaaS)». Если рядом есть поясняющий текст, писать так: «Программный комплекс предоставляется как программное обеспечение как услуга (SaaS), без передачи исключительных прав на ПО».

10Технологический стек

Актуализировать стек.

Сейчас Технологический стек: C#, TypeScript, JavaScript, .NET, ASP.NET Core, PostgreSQL, Redis, Node.js, Docker, Linux и Nginx.
Заменить на Технологический стек: C#, TypeScript, JavaScript, .NET, ASP.NET Core, PostgreSQL, Valkey, Docker, Linux и Nginx.
Менять только указанные блоки. Остальные юридические формулировки, реквизиты, ИНН, ОГРН и сведения об аккредитации не трогать.
